Styles guide (Change this content so it matches your own plan, not mine)

Background

Background is light yellow

{background-color: #FFFFCC}

Default font

Text is dark blue, 14 pixels

p { font-size: 14px; color: #003366;}

Header

The header section is 235px tall, positioned along the top of each page with the logo centered in the frame. The background is Sea Blue.

header {
height: 235px;
background-color: #336699;
background-image: url("/images/whale.png");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Navigation

The main navigation is 120px tall, positioned just below the header with a background color of Light Yellow and Dark Blue font color.

nav { height: 120px; background-color: #ffffcc; color: #003366; }

Images

Images will be over light blue background with a grey border around the container

figure { background-color: 6699CC; border: 2px solid #333333; }

img { padding: 1em; }

Links

Links will be burgundy. On hover, they will have a light blue background and a blue border.

a { color: #993333 }

a:hover { background-color: #6699CC; border 2px dotted #336699; }
